How to Set Wind Speed Step for Tunnel Measurements.

Dial up the wind speed required for your tests using the motor control potentiometer. The potentiometer scale selects the percentage of maximum motor power that will be used. The actual tunnel speed will be displayed accurately on the computer system connected to the wind tunnnel anemometer.

Before you start remember:

  • Make sure the model is mounted correctly in the test section.

  • Zero all the electronic measuring equipment.

  • Ensure that calibration settings and model
    parameters are correctly recorded in the computer system.

  • Wind Tunnel maximum speed is 100 m/s for empty test section.

  • The motor controller cannot produce a stable wind speed below 5 m/s so measurements taken in the range below this value will be inaccurate.
